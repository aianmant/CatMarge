# CatMarge 要件定義

## 概要
猫をテーマにしたスイカゲーム風マージパズルゲーム。  
小さな猫を落として同じ種類が触れるとマージ・進化し、スコアを競う。

---

## ゲームルール

### 基本ルール
- 画面上部から猫を1匹ずつ落とす
- 同じ種類の猫が接触すると合体し、1段階上の猫に進化する
- 猫が画面上部のラインを超えるとゲームオーバー
- 進化するたびにスコアが加算される

### 猫の進化チャート（全10段階）

## 画面構成

### ゲーム画面（メイン）
- **ゲームエリア**: 縦長のボックス（物理演算あり）
- **次の猫プレビュー**: 次に落ちる猫を右上に表示
- **スコア表示**: 画面上部中央
- **ゲームオーバーライン**: ゲームエリア上端

### ゲームオーバー画面
- 最終スコア表示
- リトライボタン

---

## 技術仕様

### 使用ライブラリ
| 用途 | ライブラリ |
|------|-----------|
| 物理演算 | `matter-js` |
| ゲームループ | `react-native-game-engine` |
| アニメーション | `react-native` Animated API |

### 猫オブジェクト
- 形状: 円形（シンプル実装）
- サイズ: Lv に応じて半径が増加（Lv1: 20px 〜 Lv10: 90px）
- 表示: 絵文字またはカラー円 + テキスト（初期実装）

### 物理演算
- 重力: あり（上から下へ落下）
- 衝突: 猫同士・壁・床で反発
- マージ判定: 同種の猫が接触した瞬間に発動

---

## ビジュアル方針
- ポップでカラフルな配色
- 猫ごとに異なる明るい色
- フォント: 丸みのあるもの
- SE / 演出: フェーズ2以降に追加（初期実装では省略可）
